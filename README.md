# 가구전문가 아빠와 아들 — 홈페이지

경기남부 가구이동·폐기·시스템행거 전문 출장 서비스 랜딩페이지입니다.
GitHub Pages로 배포되는 정적 웹사이트입니다.

## 📁 파일 구조 (무엇이 무엇인지)

| 파일/폴더 | 설명 | 직접 수정? |
|---|---|---|
| `index.html` | 페이지 뼈대 + SEO 정보(제목/설명/구조화데이터) | ✅ 가능 |
| `src/app.src.jsx` | **실제 화면을 그리는 원본 코드** (글자, 버튼, 가격 등) | ✅ 여기를 고침 |
| `app.js` | 위 원본을 빌드해 만든 결과물 (브라우저가 읽는 파일) | ❌ 직접 수정 금지 |
| `styles.css` | 디자인(색·여백 등) 빌드 결과물 | ❌ 직접 수정 금지 |
| `manifest.json` | "홈 화면에 추가"(앱처럼 설치) 설정 | 가끔 |
| `robots.txt`, `sitemap.xml` | 검색엔진 안내 파일 | 가끔 |
| `*.jpg`, `*.png`, `icon.svg` | 사진/로고 이미지 | ✅ 교체 가능 |
| `MANUAL.md` | **사장님이 직접 해야 할 일(검색등록·도메인 등) 안내서** | 읽기용 |

> ⚠️ **중요:** `src/app.src.jsx`를 수정한 뒤에는 반드시 아래 "빌드"를 해야 `app.js`에 반영됩니다.
> (글자만 살짝 바꾸는 정도라면, 그냥 저에게 "이 문구 바꿔줘"라고 부탁하셔도 됩니다.)

## 🔧 코드를 수정한 뒤 빌드하는 법

처음 한 번만 (도구 설치):
```bash
npm install
```

수정 후 매번:
```bash
npm run build
```
→ `app.js`와 `styles.css`가 새로 만들어집니다. 그다음 평소처럼 GitHub에 올리면(commit/push) 사이트에 반영됩니다.

## 🚀 기술 구성

- React 18 + Framer Motion (애니메이션) — **모두 `app.js`에 함께 번들**되어 외부 CDN 없이 동작합니다.
  (예전에 외부 CDN 로딩 실패로 사이트가 안 뜨던 문제를 원천 차단)
- Tailwind CSS — 빌드 시 실제 사용한 클래스만 추려 `styles.css`로 생성
- 빌드 도구: esbuild(자바스크립트), Tailwind CLI(CSS)
